# 📘 Pattern Problems in C++

## 📌 Overview

This repository contains various **pattern printing problems in C++**, organized into categories like:

* 🔷 Hollow Patterns
* 🔶 Square Patterns
* 🔺 Triangle Patterns

Each problem is implemented in a separate `.cpp` file and helps in building strong fundamentals of:

* Nested loops
* Logic building
* Pattern visualization
